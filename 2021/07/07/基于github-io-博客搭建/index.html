前言

有些同学可能一听到搭建博客就望而却步。弄个博客网站，不得有台服务器吗？不得搞数据库吗？不得注册域名吗？没事，如果都没有，那照样是能搭建一个博客的。GitHub 是个好东西啊，它提供了 GitHub Pages 帮助我们来架设一个静态网站，这就解决了服务器的问题。

首先创建一个属于你独有的github.账户

github 地址
这个不需要多讲，基本上大家去看流程就可以了。创建好账户以后进入个人主页

  • 新建一个new repository
    • 在此页面中填入你新建项目的名称。然后点击创建就可以。
    • 在新建好的项目中点击 settings
    • 拉倒最下边找到 github pages 选型点击进去
    • 然后选择一个你新建项目的分支
    • 保存
    • 可以选择一个默认的博客主题
    • 选择主题

      选择完毕之后 GitHub Pages 就会自动帮你生成好网站，在他跳转的界面点击 Commit changes 按钮，网站就可以访问了。

输入网址你账号的地址就可以访问了

如果不知道怎么查看自己的网址的话可以通过 settings -> pages 来查看

到此基于github.io 搭建博客就完毕了。但是有些人肯定会有疑问，看到网上好多别人的博客很炫，githu.io 里边的主题有些简单过于单调。如果想丰富你的页面的话，接着往下看。
